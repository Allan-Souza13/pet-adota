# 🐶 Sistema de Adoção de Pets

Projeto fullstack para gerenciamento de adoção de pets.

## 🚀 Tecnologias utilizadas

- Java + Spring Boot
- HTML, CSS e JavaScript
- API REST

## 📌 Funcionalidades

- Cadastrar pets
- Listar pets disponíveis
- Adotar pet
- Deletar pet

## 💻 Como rodar o projeto

### Backend
- Rodar aplicação Spring Boot

### Frontend
- Abrir o index.html
